# 詳細画面（pages/agent.html）仕様

- **画面ID**: `DETAIL`
- **対応実装**: `assets/app.js` `renderDetail()`
- **対応テスト**: `test/e2e/detail.spec.js`
- **前提フィクスチャ**: `test/fixtures/assets.fixture.js`（5アセット・値固定）
- **URL**: `agent.html?id=<アセットID>`

### フィクスチャの導出値

- ピック（閲覧数上位4件）= {101,102,103,104}。105 は非ピック
- id=102: コードレビューボット / 開発支援 / 閲覧800 ユニーク450 DL90 いいね60 / オーナー 佐藤 花子

---

### SPEC-DETAIL-001: 指定IDのアセット詳細（タイトル・統計・オーナー）を表示する

- **対象**: `#detail .dtl-title` / `.dtl-stat` / `.dtl-meta`
- **前提**: `agent.html?id=102`
- **挙動**: URLの id に一致するアセットの詳細を表示する
- **期待値**: タイトル=`コードレビューボット`、閲覧数統計に`800`、オーナーに`佐藤 花子`
- **テスト**: `[SPEC-DETAIL-001]`

### SPEC-DETAIL-002: カテゴリリンクは一覧へ ?cat= 付きで遷移する

- **対象**: `#detail .dtl-cat-link`
- **前提**: `agent.html?id=102`
- **挙動**: カテゴリ名リンクの遷移先が一覧画面の当該カテゴリ絞り込みになる
- **期待値**: `href` が `agents.html?cat=` を含む
- **テスト**: `[SPEC-DETAIL-002]`

### SPEC-DETAIL-003: ピック対象にはピックバッジを表示し、非対象には表示しない

- **対象**: `#detail .dtl-pick`
- **前提**: ピック対象 id=102 と 非ピック id=105
- **挙動**: `pick` が真のアセットのみバッジを描画する
- **期待値**: id=102 では `.dtl-pick` が存在、id=105 では存在しない
- **テスト**: `[SPEC-DETAIL-003]`

### SPEC-DETAIL-004: 存在しないIDでは not found メッセージを表示する

- **対象**: `#detail`
- **前提**: `agent.html?id=9999`（該当なし）
- **挙動**: id に一致するアセットが無い場合、見つからない旨と一覧への戻りリンクを表示する
- **期待値**: `アセットが見つかりませんでした。` の文言と一覧リンクを表示。`.dtl-title` は無い
- **テスト**: `[SPEC-DETAIL-004]`

### SPEC-DETAIL-005: ドキュメントタイトルがアセット名になる

- **対象**: `document.title`
- **前提**: `agent.html?id=102`
- **挙動**: 詳細表示時にページタイトルをアセット名で始まる文言に更新する
- **期待値**: `document.title` が `コードレビューボット` で始まる
- **テスト**: `[SPEC-DETAIL-005]`
