# ホーム画面（index.html）仕様

- **画面ID**: `HOME`
- **対応実装**: `assets/app.js` `renderHome()`
- **対応テスト**: `test/e2e/home.spec.js`
- **前提フィクスチャ**: `test/fixtures/assets.fixture.js`（5アセット・値固定）

|----|-------|---------|-------|---------|--------|

| 104 | 議事録要約ツール | 資料作成 | 650 | 400 | 2026-06-25 |

- 合計閲覧数 = 2,450
- カテゴリ = 資料作成(2) / 開発支援(2) / データ分析(1) の3種（初出順）

---

### SPEC-HOME-001: 統計バーに件数・閲覧数・カテゴリ数を表示する

- **対象**: `[data-stat=total]` / `[data-stat=views]` / `[data-stat=cats]`
- **前提**: フィクスチャ5件
- **挙動**: ページ読み込み時、3つの統計値を千区切りで表示する
- **期待値**: total=`5` / views=`2,450` / cats=`3`
- **テスト**: `[SPEC-HOME-001]`

### SPEC-HOME-002: プロデューサーズピックは閲覧数上位4件を表示する

- **対象**: `#picks .card`
- **前提**: フィクスチャ5件
- **挙動**: 閲覧数上位4件を「ピック」として選び、**データ順**でカード表示する（5件目以降は出さない）
- **期待値**: 4カード。集合ID = {101,102,103,104}（105は含まれない）。表示順はデータ順 101,102,103,104
- **テスト**: `[SPEC-HOME-002]`

### SPEC-HOME-003: カテゴリ索引に全カテゴリを初出順・件数付きで表示する

- **対象**: `#cat-index .cat-row`
- **前提**: フィクスチャ5件
- **挙動**: データ内での初出順に全カテゴリを並べ、各行に `N assets` の件数を出す
- **期待値**: 3行。順に 資料作成(`2 assets`) / 開発支援(`2 assets`) / データ分析(`1 assets`)。
  各行の `href` は `?cat=<カテゴリ名>` を含む
- **テスト**: `[SPEC-HOME-003]`

### SPEC-HOME-004: 人気ランキングは閲覧数降順で最大8件を表示する

- **対象**: `#ranking .rank-row`
- **前提**: フィクスチャ5件（8未満）
- **挙動**: 閲覧数の降順に最大8件を順位付きで並べる
- **期待値**: 5行。先頭行のタイトル=`コードレビューボット`、閲覧数=`800`。並び順ID=102,104,101,103,105
- **テスト**: `[SPEC-HOME-004]`

### SPEC-HOME-005: 新着は更新日降順で最大8件を表示する

- **対象**: `#newest .card`
- **前提**: フィクスチャ5件（全件 published あり）
- **挙動**: 更新日の降順に最大8件をカード表示する
- **期待値**: 5カード。並び順ID=104,102,101,103,105（先頭=議事録要約ツール）
- **テスト**: `[SPEC-HOME-005]`

### SPEC-HOME-006: カード／ランキング行クリックで詳細ページへ遷移する

- **対象**: `#picks .card`、`#ranking .rank-row`
- **前提**: `<body data-detail="page">`（index.html はページ遷移モード）
- **挙動**: カードまたはランキング行をクリックすると、詳細ページへ `?id=<アセットID>` 付きで遷移する
- **期待値**: 遷移先URLが `agent.html?id=<クリックした要素のdata-id>` を含む
- **テスト**: `[SPEC-HOME-006]`

### SPEC-HOME-007: 人気ランキングは順位が上から下へ縦に並ぶ2カラムで表示する

- **対象**: `#ranking`
- **前提**: デスクトップ幅（860px超）
- **挙動**: ランキングを2カラムのグリッドで表示する際、1位から順に1カラム目を上から下へ縦に埋め、1カラム目を使い切った順位から2カラム目を上から下へ縦に埋める（横方向に順位が並ぶ行優先配置にはしない）
- **期待値**: `#ranking` の `grid-auto-flow` が `column` であり、`grid-template-columns` が2列（等幅2値）になる。1カラム目内では上の行ほど順位が上位で、2カラム目の先頭は1カラム目の末尾の次の順位になる
- **テスト**: `[SPEC-HOME-007]`
