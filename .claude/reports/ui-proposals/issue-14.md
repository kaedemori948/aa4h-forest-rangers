# UI改善案 — Issue #14「[アセット要望] 営業提案書ドラフト生成AI」

## 概要

pages/agents.html のエージェント一覧に、新規AIアセットカードを追加する。

## 実装方針

`data/` 配下のファイルは変更禁止のため、`agents.html` の既存 `<script>` タグより前に
インラインスクリプトで `window.ASSETS_DATA.assets` に新規エントリを push することで、
`app.js` の `transformData()` に自然に取り込ませる。

## 追加カード仕様

| 項目 | 値 |
|------|-----|
| 名前 | 営業提案書ドラフト生成AI |
| カテゴリ | 営業支援 |
| 説明 | 顧客名・商材・取引履歴を入力するとWord形式の提案書ドラフトを自動生成 |
| 連携ツール | Word, PowerPoint, Salesforce |
| タグ | 提案書, 営業, ドキュメント生成 |

## 選定理由

スコア最上位案として自動選択。既存カード構造に沿った最小変更で要望を実現できる。
